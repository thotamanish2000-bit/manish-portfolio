## 🏃‍♂️ How to Run Locally

If you would like to run this customized portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/thotamanish2000-bit/manish-portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd manish-portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
